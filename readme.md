The quickest way to get weather and forecast result of the city. 

## Installation
1. Clone `git clone https://github.com/mifas/weather-api` the repo
2. run `composer install`
3. copy `.env.example` file to `.env`
4. update `OPEN_WEATHER_API_KEY`
